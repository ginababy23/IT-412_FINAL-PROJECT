# IT-412_FINAL-PROJECT

This project is a demo website for a makeup brand (LushLuxe Cosmetics) created for the IT-412 final.

Run locally (prices displayed in Philippine Peso - ₱):

```bash
npm install
npm start
# then open http://localhost:3000
```

The project exposes a simple Express server with API endpoints (prices shown in ₱):
- GET /api/products
- GET /api/artists
- POST /api/checkout
